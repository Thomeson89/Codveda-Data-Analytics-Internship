# Task 1: Regression Analysis

### Level 2(Intermediate)
## Objective:
Built a simple linear regression model using scikit-learn to predict total_spent based on quantity.
## Key Metrics:
**Coefficient:** 2.97 (Every unit increase in quantity adds 2.97 to total spend).
**R-squared:** 0.47 (47% of variance explained by quantity).
**MSE:** 18.99 (Average squared prediction error).

## Tools: 
Python, pandas, scikit-learn, matplotlib.
