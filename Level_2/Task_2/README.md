# Task 2: Time Series Analysis (Stock Prices)
**Objective:** analyzed historical stock data for Apple (AAPL) to identify long-term trends and seasonality.

## **Key Accomplishments:**
* **Data Preparation:** Filtered a 500k+ row dataset to isolate AAPL and set a proper datetime index.
* **Decomposition:** Used the statsmodels library to break the series into Trend, Seasonality, and Residuals.
* **Smoothing:** Applied 30-day and 100-day moving averages to filter market noise and visualize the core growth trend.

## **Insight Summary:**
The analysis confirmed a strong bullish trend for AAPL from 2014-2017, with a predictable annual seasonal cycle that may correlate with product release schedules.
