# Task 2: Building Dashboards with Power BI
### Level 3 (Advanced)

## Project Objective
The objective of this task was to create an interactive executive dashboard to present business insights from the Cafe Sales dataset. 
This involved importing raw, "dirty" data, cleaning it within Power BI, and designing a user-friendly interface for stakeholders.

## Technical Workflow

**Data Cleaning (Power Query):** Handled missing values, formatted date columns, and filtered out "ERROR" and "UNKNOWN" entries from the raw dirty_cafe_sales.csv.


**Visualization Design:** Developed interactive charts, including a Sales Trend Line Graph, Item Revenue Bar Chart, and a Location Distribution Donut Chart.


**Interactivity:** Implemented Slicers for Date ranges, Location, and Payment Methods to allow for dynamic data exploration.

## Key Business Insights
**Revenue Trends:** Identified peak sales periods, helping the cafe optimize staffing levels.

**Product Performance:** Coffee and Cake remain the highest revenue generators.

**Operational Split:** In-store vs. Takeaway analysis informs packaging and seating capacity needs.
